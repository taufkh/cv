# Taufik Hidayat Consulting Profile

Client-facing consulting profile built with React, TypeScript, Vite, and Tailwind CSS.

## Positioning

This site is designed as an English-first personal consulting profile for Odoo / ERP work. It emphasizes:

- structured implementation discipline
- approval and control-oriented workflow design
- Odoo 16 and 17 project exposure
- case-study-led credibility instead of generic CV presentation

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Notes

- Public case studies use a mixed disclosure model: one named case and two anonymized summaries.
- Source content for the consulting narrative lives in the section components and [`src/content/caseStudies.ts`](./src/content/caseStudies.ts).
